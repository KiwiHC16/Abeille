ab::dbVersion
    Format 'YYYYMMDD'
    Key to check if database must be updated on start.

ab::defaultGpioLib
    GPIO lib to use to control GPIO for Zigates type PI/PIv2 cases.
    Values: 'WiringPi' or 'PiGpio'

ab::defaultParent
    Default parent ID to new Abeille's equipment

ab::monitorId

ab::networkMap

ab::preventLQIAutoUpdate
    Y => Prevent LQI interrogation during night.
    N => Default

ab::preventUsbPowerCycle
    Y => Prevent USB power cycle attempt in case of Zigate not responding
    N => Default

ab::forceZigateHybridMode >>> TO BE REMOVED. Obsolete and can no longer work in 'hybrid'.
    Y => Force Zigate to use 'hybrid' mode instead of 'raw'
    N => Default

ab::gtwTypeX
    Type of gateway X.
    Values: 'zigate', 'ezsp' (NOT functional yet)

ab::gtwSubTypeX
    Sub type for gateway X.
    Values for 'zigate' type: 'PI', 'USB'...

ab::gtwEnabledX
    Indicates if gateway X is enabled or not
    Values: 'Y' or 'N'

ab::gtwChanX
    Zigbee channel for gateway X
    Values: 0=auto, 11 to ?

ab::gtwPortX
    Local port for gateway X
    Ex: /dev/ttyUSB0

ab::zgIeeeAddrOkX
    Zigate specific: Set 1 when IEEE address is validated. Used to confirm that Zigate is alive on the correct port.
