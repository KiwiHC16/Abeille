New way based on 'packages.json'
================================
If this files exists, Jeedom handles directly dependencies installation.
In that case dependancy_install() in Abeille.class is useless.
but dependancy_info() is still required but checking new paths.

Log file        : 'Abeille_packages'
In progress file: '/tmp/jeedom_install_in_progress_Abeille', contains percentage
